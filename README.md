# FastAPI_Registration
FastAPI Registration - is a registration and login module built on FastAPI.

## Requirements
* Python
* FastAPI
* Uvicorn
* Sqlalchemy

## How to install?
1. Install the required libraries (listed above)
   FastAPI
   ```
   pip install fastapi
   ```
   Uvicorn
   ```
   pip install uvicorn
   ```
   Sqlalchemy
   ```
   pip install SQLAlchemy
   ```
